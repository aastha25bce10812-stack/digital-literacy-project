# digital-literacy-project
 Digital Literacy and Online Safety

 Project Overview
This project explores the importance of **digital literacy** and **online safety** for first-year Computer Science Engineering students.  
It focuses on developing practical digital skills that are essential for academic success, professional growth, and safe participation in the online world.

The report covers five key tasks related to:
- Digital tools
- Safe internet practices
- Professional online presence
- Technical platforms
- Email etiquette
- Online payment fraud awareness

 Objectives
The main objectives of this project are:

- To understand the concept of **digital literacy**
- To explore useful **digital productivity tools**
- To learn safe and responsible **internet practices**
- To build a **professional digital portfolio**
- To practice technical skills using online coding platforms

Task 5: Online Payment Fraud and Cyber Safety
This task explored common online payment scams, especially related to **UPI fraud**.

Platforms Mentioned:
- Google Pay
- PhonePe
- Paytm

 Fraud Awareness Topics:
- Fake collect requests
- Scam calls pretending to be bank/customer support
- Suspicious payment links
- PIN sharing fraud

 Safety Measures:
- Always verify recipient details
- Never enter UPI PIN to receive money
- Avoid clicking unknown links
- Report suspicious activity immediately

 Reporting Resources:
- **National Cyber Crime Portal:** `cybercrime.gov.in`
- **Cyber Fraud Helpline:** `1930`
